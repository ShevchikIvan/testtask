<h2>Installation without docker</h2>
*The project requires node.js

Gulp is needed to be installed

<code>npm install -g gulp</code>

Then

<code>npm install</code>

To run the app on localhost:8081

<code>gulp run</code>

<h2>Docker</h2>
<code>docker-compose up</code>
